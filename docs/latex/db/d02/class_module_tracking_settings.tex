\subsection{Detailed Description}
Class providing settings for \hyperlink{class_module_tracking}{ModuleTracking}. Export 3 sliders as public members to easily connect them in \hyperlink{class_module_tracking}{ModuleTracking}. Provides labels for displaying settings values.
