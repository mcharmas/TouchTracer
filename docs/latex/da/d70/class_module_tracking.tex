Processes current frame. 

How is it done:
\begin{DoxyItemize}
\item if image is not monochromatic yet -\/ converts it
\item performs thresholding according to thres variable value (adjustable in \hyperlink{class_module_tracking_settings}{ModuleTrackingSettings})
\item searches for contours
\item filters contours using \hyperlink{class_module_tracking_a17dab2c4a2d0def3ab76158dd51eec62}{filterContours()} method
\item pushes current touches to \hyperlink{class_tracker}{Tracker} object
\item draws filtered contours on thresholded image
